# Project: Vocational Guidance Platform
## General Description

This is an API that helps students find university majors that align with their interests, skills, and personal values.

## Tech Stack

- **Backend**: .NET 8
- **Database**: PostgreSQL

## Architecture

```
/
├── API/                       # .NET API
└── API/Controllers/           # swagger endpoints
└── /DB/schema.dbml            # DB schema
└── DB/                        # BD scripts
└── IntegrationTests/          # Integration tests
```
